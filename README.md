# Kim Min-Jae Portfolio

> 안정적인 시스템을 설계하고, 테스트로 신뢰를 쌓는 개발자

Backend & AI Service 개발자 김민재의 포트폴리오 사이트입니다.

## Tech Stack

- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **Deployment**: Vercel

## Getting Started

```bash
npm install
npm run dev
```

http://localhost:3000 에서 확인할 수 있습니다.

## Project Structure

```
src/
├── app/              # 페이지 라우팅
├── components/       # UI 컴포넌트
├── data/             # 프로필, 프로젝트, 스킬 등 데이터
│   └── projects/     # 프로젝트별 데이터 파일
├── types/            # TypeScript 타입 정의
└── lib/              # 유틸리티
docs/                 # 노션 등 외부 문서용 마크다운
```
